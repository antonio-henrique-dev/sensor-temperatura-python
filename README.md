# 🌡️ Sensor de Temperatura em Python

Projeto simples desenvolvido em Python para simular leitura de temperatura e gerar alertas quando ultrapassa um limite definido.

---

## 🚀 Funcionalidades

* Leitura de temperatura simulada
* Identificação de temperatura normal e alta
* Contador de alertas
* Registro em arquivo (`log.txt`)

---

## ⚙️ Como funciona

O sistema gera valores de temperatura automaticamente e:

* Se estiver abaixo do limite → mostra "Temperatura Normal"
* Se ultrapassar → mostra "Temperatura Alta" e incrementa contador

---

## 🧪 Tecnologias usadas

* Python 3
* Biblioteca `time`

---

## ▶️ Como executar

```bash
python3 main.py
```
